Help

Quick Start
  • Add a project at the bottom of the Active tab
  • Add tasks via the quick-add bar in each project (type + Enter)
  • Click the status pill to cycle: TODO → WORKING → DONE
  • DONE tasks are automatically archived
  • Click any task to open the full edit modal
  • Pause a task with ⏸ — it moves to On Hold
  • Resume with ▶ to return it to Active
  • Reopen archived tasks with ↩ (returns as WORKING)
  • Delete sends to Trash — restore with ↩ or permanently delete
Keyboard Shortcuts
Ctrl+Z Undo Ctrl+Y Redo Ctrl+Shift+Z Redo (alternate) Ctrl+Enter Save & close modal Escape Close modal (prompts if unsaved changes) / Focus search bar Enter Submit quick-add fields
Save System
  • Green — all changes saved to file
  • Yellow — save in progress
  • Red — unsaved changes

Chrome and Edge auto-save to your file with a 1-second delay. Firefox and Safari require manual downloads. All changes are also backed up to browser storage automatically.

Save As creates a new copy. The ✕ button disconnects auto-save.

Import / Export
  • JSON Export (Settings → Data) — full backup as a .json file
  • JSON Import (Settings → Data) — load from .json or .html files
  • Markdown (Settings → Data) — copy full task list to clipboard
  • Snapshot (⎘ Export button) — read-only HTML with frozen date/time
  • New (✦ button) — blank document, preserves your settings
Tabs
  • Active — TODO and WORKING tasks, fully editable
  • On Hold — paused tasks with prior status preserved, read-only
  • Archive — completed tasks, read-only
  • Trash — deleted items, restorable or permanently deletable

All tabs show identical task detail: status, owner, badges, subtasks, and progress.

Task Features
  • Subtasks — expand with ▾, quick-add inline, independent status cycling
  • Progress — manual slider (0–100%) or auto-calculated from subtasks
  • Drag & drop — reorder tasks, subtasks, and projects via the grip handle (⠿)
  • Badges — age (fresh/warn/stale), due date, cycle time, done-vs-due
  • Collapse/Expand — per project, or use Collapse All / Expand All
Settings
  • Themes: Midnight, Light, High Contrast, Soft Dark, Cyberpunk
  • Font Size: S / M / L / XL
  • Compact Mode and Show Notes Inline toggles
  • Custom Title — replaces header text and export filenames
  • Archive Purge — auto-move old archived items to Trash
  • Trash Purge — auto-delete old trash items permanently
  • Remove deleted tasks from HTML export — when checked, the ⎘ Export snapshot will not include trash items
Print

Click ⎙ Print to generate a weekly snapshot with four sections:

  • Active Tasks — all current TODO and WORKING tasks
  • On Hold — all paused tasks
  • Recently Completed — archived in the last 7 days
  • Recently Deleted — trashed in the last 7 days

Sections with no items are omitted. Light background, no UI controls.

Tips
  • Search (/) filters by description, notes, owner, and project name
  • Undo/Redo supports up to 10 levels per session
  • All deletes go to Trash first — only permanent delete and empty trash are irreversible
  • Escape prompts before discarding unsaved modal changes
  • Collapse/expand preferences reset on reload (by design)
  • Purge intervals are configurable in Settings under Archive and Trash
